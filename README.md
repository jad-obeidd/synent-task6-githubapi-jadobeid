# 🔍 GitHub Profile Finder

A responsive web application that allows users to search any GitHub profile using the GitHub REST API and display detailed profile information.

Developed as part of the Synent Technologies Web Development Internship.

## ✨ Features

- Search any GitHub username
- Display profile picture
- View full name
- View username
- User bio
- Followers & Following
- Public repositories count
- Company
- Website
- Location
- Join date
- Display latest repositories
- Repository language
- Repository stars
- Repository forks
- Loading state
- Error handling
- Dark / Light mode
- Responsive design

## 🌐 API Used

GitHub REST API

https://api.github.com

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub REST API
- Fetch API
